# LTHW - GNU Octave

[Download GNU Octave](https://www.gnu.org/software/octave/download.html)

## Get help
```
>> help <function name>
```

```
>> help clc
'clc' is a built-in function from the file libinterp/corefcn/sysdep.cc

 -- clc ()
 -- home ()
     Clear the terminal screen and move the cursor to the upper left
     corner.
```
